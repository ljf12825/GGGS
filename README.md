GGGS(GGGameServer)

*A reusable and scalable game server*

GGGS is a lightweight game server infrastructure designed to support gameplay simulation and backend logic execution. Although my primary focus is game client development, many gameplay features require a reliable server-side environment for testing, verification, and system-level simulation. This project was created to fill that gap.

The goal of GGGS is to provide a flexible foundation for building game backend services while keeping the architecture simple enough to extend and adapt to different gameplay scenarios.
